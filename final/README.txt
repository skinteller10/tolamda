# HƯỚNG DẪN CÀI ĐẶT

1. Giải nén và upload toàn bộ file trong thư mục này lên hosting của bạn.
2. Đảm bảo hosting hỗ trợ PHP (hầu hết các host giá rẻ hiện nay đều hỗ trợ).
3. Truy cập `domain-cua-ban.com/index.php` để xem ứng dụng.
4. Truy cập `domain-cua-ban.com/admin.php` để quản lý bài viết.
   - Tài khoản Admin: skinteller.vn@gmail.com
   - (Bạn cần đăng nhập bằng mật khẩu bạn đã tạo trên Firebase).

# CƠ CHẾ HOẠT ĐỘNG
- Ứng dụng chạy trực tiếp trên trình duyệt, không cần build.
- Hình ảnh > 500KB sẽ tự động được nén trước khi upload.
- Mục 'Tớ làm da' được thiết kế đặc biệt hiển thị 1 hoặc 2 ảnh vuông.
- Dữ liệu được lưu an toàn trên Firebase Firestore.
